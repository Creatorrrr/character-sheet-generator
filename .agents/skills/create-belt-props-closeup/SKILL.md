# Create Belt Props Closeup

## Overview

Create a neutral prop and waist-accessory design reference. Avoid invented logos or unreadable fake text.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스와 동일 인물로, 허리 장식, 벨트, 파우치, 키링, 로고형 장식 등 고유 소품의 고해상도 클로즈업 이미지를 생성해줘.

구도:
- 허리 또는 가방/소품 중심의 디자인 레퍼런스 사진
- 선정적이지 않은 중립적인 의상 자료 구도
- 캐릭터의 의상 일부가 자연스럽게 포함됨

디테일:
- 가죽, 나일론, 금속, 플라스틱, 천 소재가 현실적으로 구분되게
- 버클, 지퍼, 스티치, 스크래치, 사용감
- 소품이 실제로 의상에 연결되어 있는 자연스러운 구조
- 영상 AI가 소품 모양을 유지할 수 있도록 명확하게

피해야 할 것:
floating accessories, unreadable fake text, random logos, CGI props, toy-like material, redesigned costume.
```

## Quality Check

Pass only when accessories are physically attached, readable as real materials, and do not introduce random branding.
