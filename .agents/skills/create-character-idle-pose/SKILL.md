# Create Character Idle Pose

## Overview

Create an idle pose reference for video scenes. This should be expressive enough to show character attitude but not so dramatic that it changes identity or costume.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스와 동일 인물로, 영상 제작용 캐릭터 idle pose 실사 레퍼런스 이미지를 생성해줘.

구도:
- full body 또는 knees-up character reference
- 자연스럽고 안정적인 대기 자세
- 얼굴, 헤어, 상의, 하의, 신발, 액세서리가 충분히 보이게
- 단순한 스튜디오 배경

표현:
- 캐릭터 성격이 느껴지는 자연스러운 태도
- 과장되지 않은 표정과 몸짓
- 영상 AI가 일상 대화나 대기 장면에 바로 참고할 수 있는 포즈

유지:
- 동일한 얼굴과 헤어스타일
- 동일한 의상과 소재감
- 동일한 나이대와 분위기
- 현실적인 인체 비율과 조명

금지:
anime pose, cosplay poster look, glamour pose, exaggerated body language, distorted hands, redesigned outfit, CGI, text.
```

## Quality Check

Pass only when the pose is usable for video identity reference and does not overstyle or redesign the character.
