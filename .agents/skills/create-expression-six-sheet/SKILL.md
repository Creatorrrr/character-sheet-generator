# Create Expression Six Sheet

## Overview

Create a six-panel face expression sheet with no text labels. Keep identity and camera setup consistent across all panels.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 얼굴 레퍼런스와 동일 인물로, 표정 6종 클로즈업 레퍼런스 시트를 생성해줘.

구성:
- 동일한 인물이 같은 조명과 같은 배경에서 6개의 얼굴 표정을 보여주는 레퍼런스 시트
- 정면 얼굴 클로즈업 6칸 구성
- 텍스트 라벨 없이 이미지만 정리

표정:
1. neutral expression
2. subtle smile
3. surprised
4. thinking
5. slightly embarrassed
6. confident or determined

유지:
- 얼굴형, 눈매, 코, 입술, 피부톤, 헤어스타일 완전 동일
- 의상 상단과 헤어 장식 동일
- 캐릭터 원래 성격에 맞는 감정 표현

스타일:
realistic actor expression reference sheet, live-action, natural skin texture, realistic facial muscles, no anime, no 3D render, no CGI, no plastic skin.
```

## Variations

If the user provides a character personality or asks to replace expression 6, change only expression 6. Do not introduce named character presets.

## Quality Check

Pass only when all six faces are the same person and expressions are readable without becoming cartoonish.
