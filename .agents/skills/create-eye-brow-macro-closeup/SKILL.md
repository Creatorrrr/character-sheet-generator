# Create Eye Brow Macro Closeup

## Overview

Create the eye identity macro. This image should focus on both eyes, brows, eyelashes, iris detail, and upper nose bridge.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 얼굴 레퍼런스와 동일 인물로, 눈매와 눈동자 디테일을 보여주는 초고해상도 매크로 클로즈업 이미지를 생성해줘.

구도:
- 양쪽 눈과 눈썹이 중심
- 코 윗부분까지만 살짝 보이는 극근접 클로즈업
- 고유 눈색과 눈빛을 유지
- 눈동자 반사와 속눈썹, 눈가 피부 질감이 보이게

실사 디테일:
- 실제 눈동자의 습도감
- 자연스러운 눈물막 반사
- 속눈썹의 개별 가닥
- 눈가의 미세한 피부결
- 과하지 않은 현실적 조명

목표:
영상 AI가 캐릭터의 눈매, 시선, 감정 표현을 정확히 유지할 수 있게 하는 레퍼런스.

피해야 할 것:
anime eyes, oversized eyes, artificial glowing eyes, CGI iris, doll eyes, over-sharpened lashes, plastic skin.
```

## Quality Check

Pass only when the eyes are realistic, not oversized or stylized, and both eyes belong to the same identity.
