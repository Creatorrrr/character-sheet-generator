# Create Face Front Closeup

## Overview

Create the primary master face reference. This is usually the first image in the video reference pack.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스를 기준으로, 동일 인물의 정면 얼굴 클로즈업 레퍼런스 이미지를 생성해줘.

목표:
- 영상 AI에서 캐릭터 얼굴 ID를 고정하기 위한 고해상도 정면 얼굴 자료
- 실제 사람을 촬영한 배우 프로필 사진처럼 자연스럽게
- 원래 얼굴형, 눈매, 피부톤, 머리색, 헤어스타일, 분위기를 유지

구도:
- 얼굴 중심의 정면 클로즈업
- 어깨 위까지 보이는 head and shoulders portrait
- 카메라를 똑바로 바라보는 중립 표정
- 배경은 단순한 회색 또는 흰색 스튜디오 배경

실사 디테일:
- 실제 피부 질감, 모공, 미세한 비대칭, 자연스러운 혈색
- 실제 머리카락의 가닥감과 잔머리
- 자연스러운 눈동자 반사
- 과하게 보정되지 않은 현실적인 인물 사진

피해야 할 것:
- anime, illustration, 3D render, CGI, doll-like face
- plastic skin, waxy skin, glossy AI smoothness
- 과한 뷰티 보정, 과도한 샤프닝
- 얼굴 디자인 변경, 의상 변경, 나이대 변경
- 텍스트, 로고, 워터마크
```

## Quality Check

Pass only when the portrait can serve as a stable master identity image: front-facing, photoreal, natural skin and hair, no text, and no visible redesign.
