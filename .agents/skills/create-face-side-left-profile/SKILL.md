# Create Face Side Left Profile

## Overview

Create one full side profile closeup facing image-left. Use this for side-view consistency in video shots.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 얼굴 레퍼런스와 동일 인물로, left-facing 완전 측면 얼굴 프로필 클로즈업을 생성해줘.

구도:
- side profile portrait
- 피사체의 코, 입술, 턱선, 이마, 목선이 이미지의 왼쪽을 향하게
- 어깨 위까지 보이는 구도
- 헤어 실루엣이 잘 보이게
- 배경은 단순한 회색 스튜디오 배경

목표:
영상 AI에서 고개를 돌리는 장면, 옆모습 장면, 대화 장면의 얼굴 일관성을 유지하기 위한 레퍼런스.

디테일:
- 실제 인간 얼굴 골격
- 자연스러운 코와 입술 라인
- 턱선과 목선의 현실적인 연결
- 실제 머리카락의 볼륨과 잔머리
- 고유 헤어 장식이 있다면 측면에서도 자연스럽게 보이게

금지:
anime profile, 3D model render, plastic skin, over-smoothed face, redesigned face, different hairstyle, text.
```

## Quality Check

Pass only when this is a true side profile facing image-left, not another 3/4 view.
