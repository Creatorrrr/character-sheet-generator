# Create Full Body Back Reference

## Overview

Create the rear silhouette and back-of-costume reference. Preserve identity through hair, silhouette, and costume continuity.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스와 동일 인물로, 전신 후면 실사 레퍼런스 이미지를 생성해줘.

구도:
- head-to-toe full body back view
- 자연스러운 중립 스탠딩 포즈
- 머리카락 후면 실루엣과 의상 후면 구조가 잘 보이게
- 단순한 스튜디오 배경과 바닥

유지:
- 동일한 헤어스타일, 머리색, 머리 길이와 볼륨
- 후면 의상 구조, 색상, 소재, 액세서리
- 신발과 하단 의상까지 포함
- 현실적인 착용감과 주름

목표:
영상 AI가 뒤돌아선 장면에서도 같은 캐릭터와 같은 의상으로 유지되도록 하는 기준 이미지.

금지:
redesigned outfit, random back details, floating hair, distorted body, game render, CGI, text, watermark.
```

## Quality Check

Pass only when the image is clearly a back view and costume continuity is plausible.
