# Create Full Body Front Reference

## Overview

Create the full-body front view used as the complete costume and silhouette anchor.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스와 동일 인물로, 전신 정면 실사 레퍼런스 이미지를 생성해줘.

구도:
- head-to-toe full body front view
- 자연스러운 중립 스탠딩 포즈
- 양팔과 다리 실루엣이 의상 디테일을 가리지 않게
- 단순한 스튜디오 배경과 바닥

유지:
- 동일한 얼굴, 헤어스타일, 머리색
- 전체 의상 구조, 색상, 소재, 액세서리
- 신발과 하단 의상까지 포함
- 현실적인 신체 비율과 착용감

목표:
영상 AI가 전신 구도에서 캐릭터의 실루엣과 의상을 안정적으로 유지할 수 있게 하는 기준 이미지.

금지:
anime, illustration, 3D render, CGI, mannequin body, random outfit changes, distorted limbs, text, watermark.
```

## Quality Check

Pass only when the full outfit is visible and the body reads as a real photographed person.
