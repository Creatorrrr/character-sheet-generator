# Create Hair Accessory Macro

## Overview

Create a macro detail shot centered on hair ornaments and nearby hair texture.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스와 동일 인물로, 헤어 장식만 강조한 매크로 클로즈업 이미지를 생성해줘.

구도:
- 머리카락 위의 헤어핀, 리본, 장식이 중심
- 주변 머리카락의 실제 가닥감이 함께 보이게
- 얼굴은 일부만 흐릿하게 포함되거나 생략 가능
- 얕은 심도, 실제 카메라 매크로 촬영 느낌

디테일:
- 장식의 소재감: 금속, 천, 플라스틱, 에나멜, 리본 원단 등 현실적인 질감
- 머리카락과 장식이 실제로 고정된 듯한 자연스러운 접촉감
- 스크래치, 작은 반사, 미세한 사용감

금지:
cartoon accessory, floating ornament, CGI shine, toy-like plastic, unrealistic glow, text.
```

## Quality Check

Pass only when the accessory is attached naturally to real-looking hair and does not become a floating prop.
