# Create Hair Front Detail Closeup

## Overview

Create the front hair identity detail image. The image may include part of the face, but hair structure is the subject.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 얼굴 레퍼런스와 동일 인물로, 헤어스타일과 앞머리 디테일을 보여주는 고해상도 클로즈업 이미지를 생성해줘.

구도:
- 이마, 앞머리, 양쪽 머리 라인, 헤어 장식이 잘 보이는 얼굴 상반부 클로즈업
- 얼굴 전체가 약간 포함되지만 머리카락 디테일이 중심
- 자연스러운 스튜디오 조명

디테일:
- 실제 머리카락 가닥감
- 잔머리와 자연스러운 흐름
- 머리카락 뿌리의 밀도감
- 고유 머리색과 그라데이션 유지
- 헤어핀, 리본, 장식이 있다면 현실적인 소재로 표현

피해야 할 것:
helmet-like hair, plastic hair, anime hair chunks, 3D render hair, overly perfect wig look, redesigned hair.
```

## Quality Check

Pass only when hair looks physically plausible and matches the approved character identity.
