# Create Hand Gesture Four Sheet

## Overview

Create a four-panel gesture sheet. Use this for video dialogue, reactions, and hand motion consistency.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스와 동일 인물로, 캐릭터다운 손 제스처 4종 레퍼런스 이미지를 생성해줘.

구성:
- 손과 상반신 일부가 보이는 4분할 이미지
- 같은 인물, 같은 의상, 같은 조명
- 텍스트 없음

제스처:
1. explaining gesture
2. pointing gesture
3. holding a small character-related item
4. relaxed idle hand pose

목표:
영상 AI에서 대화, 설명, 리액션 장면의 손동작을 안정적으로 만들기 위한 참고 이미지.

디테일:
- 실제 손 구조
- 자연스러운 손가락 관절
- 소매와 손의 접촉감
- 캐릭터 성격이 느껴지는 제스처

금지:
extra fingers, fused fingers, broken hands, anime hands, doll hands, 3D render.
```

## Quality Check

Pass only when all four gestures are distinct and no hand has extra, fused, or broken fingers.
