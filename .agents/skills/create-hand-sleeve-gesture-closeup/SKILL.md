# Create Hand Sleeve Gesture Closeup

## Overview

Create one closeup focused on hands and sleeve construction. For a four-panel gesture sheet, use `$create-hand-gesture-four-sheet` instead.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스와 동일 인물로, 손과 소매 디테일을 보여주는 고해상도 클로즈업 이미지를 생성해줘.

구도:
- 손, 손가락, 소매 끝, 장갑 또는 팔 장식이 중심
- 캐릭터다운 자연스러운 손동작
- 배경은 단순하고 흐리게
- 인물 전체는 나오지 않아도 됨

디테일:
- 실제 손 피부의 자연스러운 주름과 손톱
- 소매 끝의 봉제선과 원단 질감
- 장갑, 팔찌, 손목 장식이 있다면 현실적인 소재감
- 영상에서 손 제스처를 만들 때 참고할 수 있는 명확한 형태

금지:
extra fingers, distorted hands, doll hands, plastic skin, anime hands, 3D render, overly smooth skin.
```

## Quality Check

Pass only when hand anatomy is correct and the sleeve/accessory details match the approved costume.
