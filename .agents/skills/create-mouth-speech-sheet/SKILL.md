# Create Mouth Speech Sheet

## Overview

Create a six-panel dialogue reference sheet. The focus is natural mouth shapes and realistic facial muscle movement.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 얼굴 레퍼런스와 동일 인물로, 말하는 장면용 입 모양과 얼굴 표정 레퍼런스 시트를 생성해줘.

구성:
- 정면 얼굴 클로즈업 6칸
- 같은 인물, 같은 조명, 같은 배경
- 자연스럽게 대화하는 배우의 표정
- 텍스트 없음

입 모양:
1. closed neutral mouth
2. small open mouth
3. wide vowel mouth
4. soft smile while speaking
5. surprised open mouth
6. thoughtful half-open mouth

디테일:
- 실제 입술 질감
- 자연스러운 치아 노출
- 얼굴 근육의 현실적인 움직임
- 과장되지 않은 표정

금지:
cartoon mouth, exaggerated anime expression, distorted teeth, uncanny smile, plastic skin, CGI.
```

## Quality Check

Pass only when the same identity is preserved and teeth, lips, and mouth shapes are anatomically plausible.
