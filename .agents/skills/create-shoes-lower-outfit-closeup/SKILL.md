# Create Shoes Lower Outfit Closeup

## Overview

Create a lower-costume reference focused on footwear and garment details below the knee or around the feet.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스와 동일 인물로, 신발과 하단 의상 디테일을 보여주는 고해상도 클로즈업 이미지를 생성해줘.

구도:
- 무릎 아래 또는 발 중심의 의상 디자인 레퍼런스
- 신발의 형태, 굽, 끈, 장식, 소재가 명확히 보이게
- 자연스러운 스탠딩 포즈
- 단순한 스튜디오 바닥

디테일:
- 실제 신발 소재감
- 밑창, 끈, 버클, 금속 장식, 원단 주름
- 착용감과 현실적인 무게감
- 의상 하단의 색상과 실루엣 유지

금지:
overly stylized boots, game asset render, floating feet, distorted legs, glossy plastic, CGI.
```

## Quality Check

Pass only when shoes and lower outfit match the approved design and foot/leg anatomy is stable.
