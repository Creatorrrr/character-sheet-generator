# Create Upper Costume Closeup

## Overview

Create a non-sensual wardrobe reference focused on the neck, shoulders, collar, and upper garment construction.

## Prompt

```text
첨부한 실사화 캐릭터 시트 또는 승인된 마스터 레퍼런스와 동일 인물로, 상의의 카라, 목 장식, 리본, 재킷 상단 디테일을 보여주는 고해상도 의상 클로즈업을 생성해줘.

구도:
- 목부터 어깨, 가슴 윗부분까지만 보이는 의상 중심 클로즈업
- 얼굴 하관은 살짝 포함 가능
- 캐릭터의 상징적인 상의 구조와 색상 유지
- 선정적이거나 과장된 구도 없이 의상 디자인 자료처럼 촬영

디테일:
- 실제 원단의 직조감
- 봉제선, 단추, 지퍼, 리본, 금속 장식
- 옷의 두께감과 자연스러운 주름
- 착용 중 생기는 미세한 그림자와 접힘

스타일:
realistic costume design reference photo, live-action wardrobe close-up, natural fabric texture, no CGI, no illustration, no redesigned outfit.
```

## Quality Check

Pass only when the garment is realistic, non-sensual, and faithful to the approved costume design.
